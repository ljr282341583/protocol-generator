# 人民调解协议生成器

> 纯前端HTML单页面应用，即开即用，自动生成规范的人民调解协议书及全套配套卷宗文书。

## 快速开始

1. 选择对应方案的 HTML 文件
2. 用浏览器打开即可使用
3. 无需安装任何依赖，无需服务器

---

## 版本与方案

| 文件夹 | 授权方式 | 版本 | 说明 |
|--------|----------|------|------|
| `方案一启动码/` | 启动密码 | v5.1 | 每次打开输入启动密码，适合内部使用 |

| `密码授权版/` | 启动密码（变体） | v5.1 | 方案一密码授权变体版本 |

## 核心功能

### 协议生成
- 多种预设纠纷类型（邻里纠纷、婚姻家庭、损害赔偿、合同纠纷、劳资纠纷、房屋宅基地等）
- 当事人信息录入（支持个人/单位切换）
- 自动/手动生成协议书编号
- 可自定义调解协议条款
- 事实与理由支持模板导入导出、合并填写

### 文书导出
- **全套卷宗文书**（12种）：申请书、受理登记表、调查记录、调解记录、协议书、送达回执、权利义务告知书、口头协议登记表、回访记录、司法确认申请书、卷宗封面、情况说明
- **个人卷宗文书**（8种）：卷宗标准封面、新卷内目录、回访记录、登记单、情况说明、收据、谅解书、未成年情况说明
- 支持 Word 导出（.doc 格式，宋体/仿宋字体，A4 页面设置）
- Word 原生分页符，每个文书独立分页
- HTML 预览、复制文本、直接打印

### 模板管理
- 纠纷类型模板 JSON 导入/导出
- 支持自定义纠纷类型和条款模板

## 授权方案对比

| 特性 | 方案一（启动密码） | 方案二（授权码 + 设备绑定） |
|------|-------------------|---------------------------|
| 授权方式 | 启动密码 | 授权码 + 设备指纹 |
| 设备绑定 | 无 | Canvas/WebGL 指纹，SHA-256 哈希 |
| 云端存储 | 无 | Supabase |
| 多设备 | 不限制 | 一码一设备（可配置上限） |
| 远程管理 | 无 | 管理员面板远程撤销/重新激活 |
| 定时校验 | 无 | 30秒定时检查授权状态 |
| 离线回退 | N/A | Supabase 不可用时回退 localStorage |
| 适用场景 | 内部使用 | 对外分发，需管控授权 |

## 技术栈

- **UI**：Tailwind CSS（CDN）
- **后端**：Supabase（方案二云端绑定）
- **运行环境**：纯浏览器端，零依赖安装

### Supabase 配置（方案二）

| 项目 | 值 |
|------|-----|
| 数据表 | `device_bindings` |
| 字段 | `codeHash`, `devices`, `maxDevices`, `activatedAt`, `updatedAt` |
| 认证 | 匿名登录 |
